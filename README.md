# 📊 Sales Data Analysis Using MySQL

## 📌 Project Overview
This project demonstrates an end-to-end SQL workflow using MySQL, including database creation, automated data generation, and business-focused data analysis.
The goal of this project is to simulate a real-world sales environment and extract meaningful business insights using structured queries.


## 🛠 Tools & Technologies

- MySQL
- SQL (Data Definition & Data Analysis)
- Stored Procedures
- Aggregation & Grouping Functions

---

## 🗂 Project Structure
This project includes:

- Database creation
- Table design
- Automated generation of 100 randomized sales records
- Business analysis queries

The entire project is fully reproducible using the provided SQL script.

---

## 🚀 How to Run This Project

1. Open MySQL Workbench.
2. Open the file:  
   `sales_project_full_analysis.sql`
3. Run the script.

The script will:
- Create the database
- Create the sales table
- Generate 100 randomized sales records
- Execute business analysis queries

---

## 📈 Business Questions Answered

This project answers key analytical questions such as:

- What is the total revenue generated?
- Which product categories generate the most revenue?
- What are the top 5 revenue-generating products?
- Who are the top 5 customers?
- What are the monthly revenue trends?
